# STM32_DUAL_COUNTER_SYSTEM
An STM32-based embedded application implementing a dual-counter system with persistent memory and dual operational modes, featuring real-time seven-segment display control, user input handling, and non-volatile data retention using Embedded C and the STM32 HAL library.

## Hardware Specifications

| Component | Specification |
|-----------|--------------|
| Microcontroller | STM32G030F6P6 (ARM Cortex-M0+) |
| Development Environment | STM32CubeIDE |
| Configuration Tool | STM32CubeMX |
| Programming Language | Embedded C |
| Framework | STM32 HAL Library |
| Display | Seven-Segment Display |
| User Input | Push Buttons |
| Non-Volatile Memory | Internal Flash Memory |
| Debug Interface | SWD |
| Power Supply | 3.3V |
